Mengembangkan sistem kontrol PID berbasis closed-loop untuk mengatur suhu panel surya dengan memanfaatkan ESP32, sensor DS18B20, 
INA219, BH1750, serta driver L298N untuk integrasi dan pengendalian sistem, sehingga berhasil meningkatkan kinerja sistem melalui 
peningkatan daya keluaran dan efisiensi yang lebih stabil.
